# Sorting Hat
This project is about sorting hat. On the website, it provides introduction to Hogwart shcool and clicking let the magic begin, gives the first year student form. You need to enter Student name and sorting it will display student card. It shows your name, house and Expel button. Also clicking on Expel, delete the student card. This project is exploration of event listener and functions.

## How to run this project
* Use npm to install http-server in your terminal:
```
npm install -g http-server
```
* Run the server
```
hs -p 9090
```
* Open chrome and navigate to :
```
localhost:9090
```
